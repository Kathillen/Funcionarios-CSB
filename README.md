# Descrição
  Este projeto é uma reformulação de um projeto anterior meu, o 'Aranhaverso'. Agora, ele conta com mais dados a serem coletados e manipulados. O sistema está aprimorado, e os bugs encontrados no 'Aranhaverso' não foram identificados neste novo projeto.

  Estou muito feliz com minha evolução na programação. Há alguns meses, o tradicional console.log('Hello World!') já era algo surpreendente para mim. Estou colocando em prática meus aprendizados do curso de análise de dados.

  A cada dia, fico mais surpresa com o que o conhecimento e o estudo podem me proporcionar.

O futuro está cada vez mais perto.
